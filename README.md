# bootstrap-project
Bootstrap 5 Remix Project
# Project Overview
This project was built using Bootstrap 5 CDN. It includes:
- Home page
- About page
- Contact page
- Fully responsive layout

## Steps Followed
1. Studied Bootstrap documentation
2. Extracted Navbar, Cards, Grid, and Forms
3. Built pages step-by-step
4. Tested responsiveness
5. Deployed using GitHub Pages

## Tools Used
- HTML5
- Bootstrap 5
- Notepad
- vs code
- Git & GitHub
- ChatGPT (for guidance)

## Challenges Faced
- Navbar collapse issue (fixed by adding JS bundle)
- Spacing alignment problems (solved using Bootstrap utilities)

## Learning Outcome
- Better understanding of Bootstrap grid
- Improved UI layout skills
- Learned deployment process
